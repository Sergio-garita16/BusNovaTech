Requerimientos implementados:
-Modulo 1.0
*Módulo de Autenticación (`Login`):Control de accesos mediante consola interactiva (`JOptionPane`) con un sistema de límite estricto de 3 intentos para salvaguardar la seguridad del operador.
*Cola de Prioridad Inteligente (`ColaPrioridad`):Algoritmo de ordenamiento dinámico que organiza la fila de los pasajeros de manera automática al ser insertados, aplicando criterios integrados de edad y tipo de servicio contratado
*Estructuras de Datos Propias:Modelado dinámico de la información sin depender de librerías de colecciones genéricas de Java, optimizando el uso de memoria mediante nodos enlazados.
